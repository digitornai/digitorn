# Client SDKs

| Client | Page | Use it for |
|--------|------|-----------|
| React preview SDK | preview-sdk.md | The `@digitorn/preview-sdk` npm package. Hooks and components used inside `web/` of any preview app to subscribe to workspace state. |

## Choosing an SDK

If you are **shipping a Lovable-style web preview** (an in-app
React sandbox the agent writes to), use the React preview SDK
inside `web/` so the user sees state stream in.
