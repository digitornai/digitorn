# Software procedures

## Application install requests
- Standard catalog apps (browser, PDF reader, Office) are self-service from the
  Company Portal. Point the user there first.
- Non-catalog software needs a license check and manager approval before install.

## Office / mail issues
- "Mailbox full": guide the user to archive items older than 12 months; quota is
  50 GB. Do not raise quota from a ticket — that needs IT Ops approval.
- Repeated credential prompts in Outlook: clear cached credentials in the
  Credential Manager, then restart Outlook.

## Crashes after an update
1. Confirm the OS build is current.
2. Repair the app from Settings → Apps.
3. If it persists, capture the crash event ID from Event Viewer and attach it.

## Escalation
License purchases, anything requiring admin rights on the machine, or data-loss
situations are escalated to Software L2.
